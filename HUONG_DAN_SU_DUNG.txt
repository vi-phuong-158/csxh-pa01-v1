HƯỚNG DẪN SỬ DỤNG PHẦN MỀM SECURITY PROFILE 360 (OFFLINE)
==========================================================

1. Giới thiệu
   - Đây là phiên bản đóng gói chạy offline, không cần cài đặt Python hay Internet.
   - Thư mục chứa phần mềm có thể copy sang máy khác để chạy ngay.

2. Cách khởi động
   - Vào thư mục `dist/csxh_pa01`.
   - Tìm file `csxh_pa01.exe` (có biểu tượng ứng dụng).
   - Nhấn đúp chuột để chạy.
   - Một cửa sổ đen (terminal) sẽ hiện lên -> Đừng tắt cửa sổ này.
   - Trình duyệt sẽ tự động mở trang web ứng dụng (thường là http://localhost:8501).

3. Lưu ý quan trọng
   - Không xóa các file/thư mục đi kèm trong cùng thư mục với file .exe.
   - Dữ liệu (file db, excel) sẽ được lưu tại thư mục chứa file .exe.
   - Để tắt phần mềm: Chỉ cần đóng cửa sổ đen (terminal).

4. Xử lý sự cố
   - Nếu trình duyệt không tự mở: Hãy mở Chrome/Edge và gõ `http://localhost:8501`.
   - Nếu dữ liệu không lưu: Kiểm tra xem thư mục có bị đặt ở chế độ "Chỉ đọc" (Read-only) không.
