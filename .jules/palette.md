# Palette's Journal - Critical UX Learnings

## 2024-05-22 - Search to Create Flow
**Learning:** Users often search for an entity before creating it. If the search yields no results, the transition to the "Create" flow should be seamless and context-aware.
**Action:** Implement "Not Found -> Add New" buttons in search results that navigate to the creation form and pre-fill the search query (e.g., CCCD or Name) to reduce friction and data entry errors.
